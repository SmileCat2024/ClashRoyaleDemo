# 官方原版音频素材库

此目录保存从 `resource/音效` **子文件夹**及额外提供的气球兵、重甲亡灵、`后续音频`、`后续觉醒`、`音频` 目录复制而来的原始素材。当前有 300 个 WAV，以及 20 个 FMOD `.bank/.fsb` 文件；根目录零散文件未纳入。竞技场、主题模式和 2v2 专属 jingle 已按需求排除。文件按原始子文件夹名保留，便于追溯来源，且不会覆盖 `sfx/`、`bgm/` 内已有的旧资源。

- `1`：加载、结算与通用短音乐（竞技场/主题模式部分已排除）
- `2`：战斗 UI、皇冠、倒计时、圣水与阶段提示
- `3`：大厅音乐
- `4`：常规战斗、加时战斗音乐
- 其余目录：角色或卡牌专属音效
- `后续音频`：王子、火球、巨人、女武神、弓箭手、火枪手、小皮卡、野猪骑士；其中 `.bank/.fsb` 仅归档，Godot 不直接引用
- `后续觉醒`：觉醒火枪手与觉醒迫击炮；其中 `.bank` 仅归档，Godot 不直接引用
- `音频`：万箭齐发、哥布林、圣水收集器、大皮卡、毒药、迫击炮；其中 `.bank` 仅归档，Godot 不直接引用

运行时只引用经过确认并转码的 `.ogg` 副本；除骑士、觉醒骑士、公主、神箭游侠、气球兵、重甲亡灵、地狱塔和 BGM 外，后续已接入王子、火球、巨人、女武神、弓箭手、火枪手、小皮卡、野猪骑士及两张现有觉醒牌的可明确对应动作音。本轮 `音频` 目录的 30 个 WAV 均已接入相应卡牌或系统事件。
